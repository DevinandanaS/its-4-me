Project Description
its-4-Me empowers you to take control of your time.  Create your ideal daily schedule, then track your actual activities.  Our app visually compares the two, highlighting discrepancies and providing personalized suggestions to help you stay on track and achieve your goals.
The Problem statement
Balance the work life
The Solution
we build a web app that organises users schedule according to his/her preference
